# Project Management Skill

## Context
This skill enables the agent to effectively manage the MAS Dashboard project lifecycle, ensuring documentation is up-to-date and tasks are tracked accurately.

## Capabilities
- **Task Tracking:** Ability to update `task.md` and check off completed items.
- **Documentation:** Generate and maintain `PROJECT_MEMORY.md`, `README.md`, and technical plans.
- **Status Reporting:** Create concise status updates and walkthroughs for the user.
- **Folder Structure:** Enforce the usage of `project_docs/` for all documentation artifacts.

## Guidelines
1. Always update `PROJECT_MEMORY.md` after a significant milestone.
2. Keep `task.md` in sync with the actual progress.
3. Use Turkish for all documentation content as per user preference.
