# Frontend Skill

## Context
This skill covers the creation and maintenance of the visual interface, ensuring a high-quality "Mission Control" aesthetic.

## Capabilities
- **Component Development:** Create modular React components (Sidebar, Feed, Stats) using TypeScript.
- **Styling:** Utilize Tailwind CSS v4 with the custom MAS color palette (Deep Forest Green, Neon Green).
- **Animation:** Implement smooth transitions and effects using Framer Motion.
- **State Management:** Manage global application state with Zustand.

## Design Standards
- **Theme:** Dark Mode default.
- **Colors:**
  - Background: `var(--color-mas-green)`
  - Accent: `var(--color-mas-light)`
  - Panel: `var(--color-mas-gray)`
  - Memory: `var(--color-mas-memory)`
- **Typography:** Sans-serif, clean, data-oriented fonts.

## Guidelines
1. Prioritize visual feedback (hover states, active indicators).
2. Ensure the UI is responsive and clutter-free.
3. Use `lucide-react` for consistent iconography.
